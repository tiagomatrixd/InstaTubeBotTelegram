# InstaTubeBotTelegram

InstaTubeBotTelegram é um bot para o Telegram que permite baixar vídeos e músicas do YouTube, YouTube Music, TikTok, Instagram, X e TikTok. Basta enviar o link do vídeo ou música, e o bot reconhecerá e enviará o conteúdo diretamente no grupo do Telegram.

## Funcionalidades

- Baixar vídeos do YouTube, TikTok, Instagram e X.
- Baixar músicas do YouTube Music.
- Enviar o vídeo ou música diretamente no grupo do Telegram.
- Suporte para múltiplas plataformas de conteúdo.

## Como Usar

1. **Inicie uma conversa com o bot no Telegram:**
   Encontre o bot no Telegram e inicie uma conversa.

2. **Envie o link do vídeo ou música:**
   Envie um link de um vídeo ou música do YouTube, YouTube Music, TikTok, Instagram ou X.

3. **Receba o conteúdo:**
   O bot reconhecerá o link e enviará o vídeo ou música diretamente no grupo do Telegram.

## Tecnologias Utilizadas

- [Node.js](https://nodejs.org/)
- [Telegraf](https://telegraf.js.org/)

## Instalação

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/tiagomatrixd/InstaTubeBotTelegram.git
   cd InstaTubeBotTelegram


2. **Instale as dependências**
    ```bash
    npm install

3. **Configure as variáveis de ambiente: Crie um arquivo .env e adicione suas chaves de API e tokens do Telegram**
    ```bash
    BOT_TOKEN=your_telegram_bot_token

4. **Inicie o bot**
    ```bash
    npm start
